How to Setup Git

This guide is to help everyone set up Git on their machine. Please follow each step carefully. If you have any questions, please message either Holden or Magiber on Discord. Make sure that this is the first How To Guide you follow.

Instructions:

  1. Download GitHub Desktop: GitHub Desktop
  2. Download Git LFS (Git Large File Storage): Git LFS
  3. Make sure to have sent either Holden or Magiber your GitHub username so that we can add you to the organization. You can see your organization by clicking your profile on the top right.

Come back to this part after you finish the other two How To Guides:

  1. How to Set Up Visual Studio
  2. How to Set Up Unity
  1. Go to the DTDGameDev repository: DTDGameDev Repository
  2. Click on the assigned repository, and from there click on the green code button.
  3. Click on the “Open with GitHub Desktop”.
  4. You will be prompted to clone this repo to your machine. Please make sure to create a separate folder in a place you will be able to remember and clone the repo into that folder.
  5. Make sure you're on the main branch, and click on the “Fetch Origin” to make sure you are on the latest version.
  6. Go to your Unity Hub, on the top right you will see the add button, click on it. Double click on the folder you created and click on the folder inside of the folder you created. This will add the project onto your Unity Hub.
  7. Double click the project on Unity Hub and let it build. This will take a long time if this is your first time building the project. It will get significantly faster to load later on.
