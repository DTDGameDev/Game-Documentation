How to Setup Visual Studio

This document is to detail on how to get Visual Studio onto your local machine and to make sure it is properly set with the correct plugins.

Instructions:

  1. Downloading Visual Studio on your device, it is recommended to create a custom folder to put all of your Unity downloads.
    1. Download Visual Studio 2022 using this link: Visual Studio 2022. Open up the setup Visual Studio file you downloaded and let it run (make sure you have it downloaded in a file you can easily find).
    2. Make sure to have these two plugins check-marked so that you can download them at the same time (C# development, Game Development using Unity, and .Net).
